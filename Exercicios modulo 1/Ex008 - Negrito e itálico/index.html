Principais formatação

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag B (Não semântica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (Semântica.)

Itálico / Ênfases

Nesta frase, temos um termo em itálico usando a tag I (Não sêmantico)

Nesta frase, temos um termo em ênfases usando a tag EM (sêmantica).

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande com a tag BIG, também estamos criando um texto pequeno com a tag SMALL.

Texto deletado

Podemos marcar um texto com excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfases e indicar que ele foi adicionado depois. Utilizando a tag INS.

Existe também um texto sublinhado com a tag U (NÃO SÊMANTICO).

Texto sobrescrito

Para inserir coisas do tipo X20+3 Utilizando a tag SUP.

Texto Subscrito

Para inserir coisas do tipo H2O, utilizando a tag SUB.
